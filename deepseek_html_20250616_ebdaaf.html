-- SQL语句将实时显示在这里
-- 支持高亮语法显示

-- 示例:
CREATE USER dev_user IDENTIFIED BY "Str0ngP@ss!";

GRANT CREATE SESSION, CREATE TABLE TO dev_user;

GRANT SELECT, INSERT ON employees TO dev_user;
GRANT SELECT ON departments TO dev_user;

使用说明

  • 输入用户名和密码（密码需包含大小写字母、数字和特殊字符）
  • 选择需要授予的系统权限
  • 输入表名或视图名（多个用逗号分隔）
  • 选择对象权限（SELECT/INSERT/UPDATE/DELETE）
  • 点击"生成SQL语句"按钮创建SQL
  • 右侧面板实时显示生成的SQL语句
  • 点击"复制到剪贴板"按钮复制SQL
SQL已复制到剪贴板！
